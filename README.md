🌌 Kiana Hyprland Config
Custom automated setup optimized for low-end hardware (Dell Latitude 2011).

🔥 The Main Feature
The core of this rice is a custom color-sync script. It uses pywal to instantly match the colors of:

Hyprland (Borders/UI)

Telegram Desktop

Brave Browser (Start page)

Terminal & GTK

🛠️ How to use
Clone the repo:

Bash
git clone https://github.com/ShadowCvrsxd/.files.git
Run the switcher:

Bash
./scripts/color-switcher.sh
